\# Таймер на C++



\## Описание

Консольный таймер обратного отсчёта с поддержкой разных форматов ввода времени.



\## Возможности

\- Поддержка форматов ввода:

&#x20; - `5` — 5 секунд

&#x20; - `1:30` — 1 минута 30 секунд

&#x20; - `1:30:45` — 1 час 30 минут 45 секунд

\- Прогресс-бар с процентами

\- Отображение оставшегося времени в формате ЧЧ:ММ:СС

\- Таймер работает в отдельном потоке

\- Модульные тесты (Google Test)



\## Сборка и запуск



```bash

mkdir build

cd build

cmake ..

cmake --build . --config Release

cd Release

timer\_app.exe
